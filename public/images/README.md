# Drop image assets here

These files are referenced by `src/data/site.ts`. Until a file exists, the site
shows a styled gradient placeholder in its place (no broken images), so you can
deploy now and add the images whenever.

Expected files (from `sws-assets.zip`):

| File | Used in |
| --- | --- |
| `promo.jpg` | Hero featured image (with the play button) |
| `course.jpg` | Flagship course image + "The Course" venture card |
| `socialhippi.jpg` | Ventures → Social Hippi |
| `nerige.jpg` | Ventures → Nerige Story |
| `startup.jpg` | Ventures → Startup With Shashank |
| `trips.jpg` | Ventures → Trips To Luxury |
| `book.jpg` | Ventures → The Book |
| `tenmin.jpg` | Ventures → 10 Minutes Friend |
| `mentorship.jpg` | Ventures → 1-to-1 Mentorship |
| `portrait.jpg` | (reserved for an About/founder section) |
| `collage.jpg` | (reserved for an "at a glance" section) |

Just copy the unzipped `.jpg` files into this folder (`public/images/`) and
rebuild / redeploy. No code changes needed.
